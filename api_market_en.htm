Quotes /v2/market/* Public quote class interface, including transaction, depth, quotes, etc.

Request Format

There are currently only two methods: GET and POST

  • GET request: all the parameters are in the path parameter
  • POST request, with all parameters sent in form-data format in the body of the request

Return Format

All interfaces are in JSON format. There are three fields at the top of JSON: msg, code and data. The first two fields represent the request status and information, the actual business data is in the data field .

Here is an example of a return format:

{
  "msg": "success",
  "code": "0",
  "data": // per API response data in nested JSON object
}
. .
Parameter Name Data Types Description
msg string API information returned
code int Interface status returned, 0 represents success
data object Interface returns data subject

Public Information Interface

Contracts

Endpoint B1 provides a summary of every single contract traded on the exchange.

API Key Permissions: no signature required.

Limit (according to IP limit): 50 times / s.

HTTP request

  • GET /fapi/v2/market/contracts
curl "https://fapi.bkex.com/fapi/v2/market/contracts"

Contracts Specification

Describes the specification of the contracts

API Key Permissions: no signature required.

Limit (according to IP limit): 50 times / s.

HTTP request

  • GET /fapi/v2/market/contractSpecifications
curl "https://fapi.bkex.com/fapi/v2/market/contractSpecifications"

No parameters

Response:

            
{
    "code": 0,
    "msg": "success",
    "data": [
        {
            "contract_type": "1000shib_usdt",
            "contract_price": "0.012364",
            "contract_price_currency": "usdt"
        },
        {
            "contract_type": "sol_usdt",
            "contract_price": "33.05",
            "contract_price_currency": "usdt"
        }
    ]
}

Response Data

Parameter Name Data Types Examples Description
contract_type string btc_usdt Transaction pair
contract_price int "0.012364" Describes the price per contract.
contract_price_currency string "usdt" Describes the currency which the contract is priced
